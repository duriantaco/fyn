# [First steps with fyn](#first-steps-with-fyn)

After [installing fyn](../installation/), you can check that fyn is available by running the `fyn` command:

```
$ fyn
An extremely fast Python package and project manager.

Usage: fyn [OPTIONS] <COMMAND>

...
```

You should see a help menu listing the available commands.

## [Next steps](#next-steps)

Now that you've confirmed fyn is installed, check out an [overview of features](../features/), learn how to [get help](../help/) if you run into any problems, or jump to the [guides](../../guides/) to start using fyn.
