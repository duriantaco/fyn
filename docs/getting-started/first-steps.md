# First steps with fyn

After [installing fyn](./installation.md), you can check that fyn is available by running the `fyn`
command:

```console
$ fyn
An extremely fast Python package manager.

Usage: fyn [OPTIONS] <COMMAND>

...
```

You should see a help menu listing the available commands.

## Next steps

Now that you've confirmed fyn is installed, check out an [overview of features](./features.md),
learn how to [get help](./help.md) if you run into any problems, or jump to the
[guides](../guides/index.md) to start using fyn.
